## Description
This project is designed to precisely remove watermarks from Sora 2 generated videos using AI. It provides an intelligent solution for detecting and removing watermarks frame by frame using advanced inpainting techniques.

## How it Works
This project uses AI to remove watermarks from Sora 2 generated videos. When a user uploads a video, the system analyzes each frame to detect the watermark and removes it using inpainting. The processed video is then rendered and returned to the user.

## Installation
Before using the watermark remover, you need to set it up on your machine. Clone the repository using Git:

```bash
git clone https://github.com/shivas1432/sora2-watermark-remover.git
cd sora2-watermark-remover
```

Then install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

### Desktop Version
To start using the program, launch the main.py file:

```bash
python main.py
```

This will open a web page in your browser where you can upload your videos. The processed results will be saved in the output folder.

### Web Server Version
You can also use the web server version for your projects. Run **server.py:**

```bash
python server.py
```

The web server will start on port 8000.
